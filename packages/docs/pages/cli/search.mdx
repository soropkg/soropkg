# soropkg search

Search the soropkg registry by name, keyword, or category.

```bash
soropkg search <query> [flags]
```

---

## Arguments

| Argument | Required | Description |
|---|---|---|
| `<query>` | Yes | Search term ‚Äî matches package name, description, and tags |

---

## Flags

| Flag | Description |
|---|---|
| `--category <cat>` | Filter by category (e.g. `lending`, `dex`, `oracle`) |
| `--network <net>` | Filter to packages available on `mainnet`, `testnet`, or `futurenet` |
| `--json` | Output raw JSON |

---

## Example

```bash
soropkg search blend
```

```
blend-capital/blend@2.0.1
  Lending protocol core contracts
  Categories: lending, defi
  Networks: mainnet, testnet

blend-capital/backstop@2.0.1
  Backstop module for the Blend protocol
  Categories: lending, defi
  Networks: mainnet, testnet
```

---

## Browse all packages

```bash
soropkg search ""
```

Returns all packages in the registry ordered by most recently published.

---

## Categories

Current categories in the registry:

| Category | Description |
|---|---|
| `lending` | Lending and borrowing protocols |
| `dex` | Decentralised exchanges and AMMs |
| `liquidity` | Liquidity management |
| `stablecoin` | Stablecoin protocols |
| `oracle` | Price oracles |
| `yield` | Yield aggregators |
| `rwa` | Real-world asset protocols |
| `standard` | Stellar standard contracts (token, etc.) |
